== Description ==

Auranet Recent Photos exibe as fotos mais recentes de um usuário do Instagram previamente configurado em seu site Wordpress. Use [instagram]

== Screenshots ==

1. The configuration options for the thumbnail display.
2. The display of thumbnails on the configured page. The image is uploaded live and shown as it is currently on the site.
